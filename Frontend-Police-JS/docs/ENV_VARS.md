# Frontend Environment Variables

Create a `.env` file in `Frontend-Police-JS/` (this file is not committed to Git).

Required:

- `VITE_API_BASE_URL`
  - Example (local backend): `http://localhost:8000/api/v1`
- `VITE_GOOGLE_MAPS_API_KEY`
  - Your Google Maps JavaScript API key for the map UI

Example `.env`:

VITE_API_BASE_URL=http://localhost:8000/api/v1
VITE_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here
